We‚Äôve spent some time getting to know VS Code, and VS Code is an excellent text editor and can be used to edit remote files, but being able to use a command-line text editor is useful because command-line text editors are always there.

Editing and saving a file

Open vim again (if didn‚Äôt close it, quit, then open it again), but this time pass it a filename to create a new file with that name.

Once you‚Äôre in vim, enter insert mode by typing i on your keyboard. Now start typing!

Once you‚Äôre finished, press Esc on your keyboard to get back into normal mode.

Copying and pasting into vim

You can create new text files on the command-line with vim, but sometimes you also want to copy some text from one place and then paste it into vim, just like you do with other programs.

You can paste into vim, but you have to tell vim that you‚Äôre about to paste something into it. You have to tell vim that you‚Äôre going to paste something into it so that vim knows not to do things like automatically indent when you‚Äôre editing a file with code in it.

We tell vim that we‚Äôre going to paste something into it by ‚Äúsetting a flag‚Äù called ‚Äúpaste‚Äù.

Just before you paste something in vim, enter normal mode (press Escape on your keyboard), then type :set paste and press Enter.

Now re-enter insert mode (press i on your keyboard) and paste whatever you wanted to paste.

Finally, enter normal mode, then type :set nopaste and press Enter.

OK, but how do we paste?

Copying and pasting in Windows

You can copy in Windows by highlighting text, then either:

  • Use menus: Edit ‚Üí Copy, or
  • Use keyboard shortcuts: press Control+C on your keyboard.

To paste text into your terminal window you can do one of two things:

  • Use your mouse: Right-click, or
  • Use keyboard shortcuts: press Control+Shift+V. Notice that different from the normal shortcut for pasting!
Copying and pasting in macOS

You can copy and paste in macOS by highlighting text, then either:

  • Use menus: Edit ‚Üí Copy, or
  • Use keyboard shortcuts: press Command+C on your keyboard.

To paste text into your terminal window you can do one of two things:

  • Use menus: Edit ‚Üí Paste, or
  • Use keyboard shortcuts: press Command+V on your keyboard.
Copying and pasting in Linux

How you copy and paste may depend on the Linux distribution you‚Äôre using and the desktop environment that you‚Äôre using.

Assuming that you‚Äôre using Ubuntu with the default Gnome interface, you can copy and paste text by highlighting some text, then either:

  • Use menus: Edit ‚Üí Copy, or
  • Use your mouse: Right-click and select Copy, or
  • Use keyboard shortcuts: press Control+C on your keyboard.

To paste text into your terminal window you can do one of two things:

  • Use menus: Edit ‚Üí Paste, or
  • Use your mouse: Right-click and select Paste, or
  • Use keyboard shortcuts: press Control+Shift+V. Notice that different from the normal shortcut for pasting!

Cheatsheets

Despite its minimal appearance, vim is a full-featured text editor that can do everything that you might want. As you‚Äôve seen, interacting with programs on the command line is done entirely through your keyboard (you can keep clicking on the terminal, but all you‚Äôre doing is making fun clicking or tapping sounds).

When you‚Äôre in normal mode, you can get vim to do many different things for you, but listing out all of those different things would be painful and tedious.

You should seek out and find a ‚Äúcheat sheet‚Äù for vim that lists a good set of things that you can do with vim in normal mode. Some examples of operations that you‚Äôre going to want to do with vim frequently and should be included in the cheat sheet are:

  • Copying or cutting and pasting lines from one place to another (y to yank/copy, d to cut, p to paste).
  • Undoing or redoing changes to a file (u to undo, Control+R to redo).
  • Moving around on lines or around the file (G to go to the bottom of the file, gg to move to the top).
  • Doing the same thing multiple times (cutting the next three lines d3‚Üì).

Other editors

vim is not the only text editor, and you‚Äôre not required to use vim for this course (or any course, really). You‚Äôre welcome to try out and use other text editors, like nano, pico, or emacs.
